# Basic College Chatbot

A friendly, lightweight chatbot that helps students find college information (courses, services, schedules, contacts). Built as a teaching/demo project to show how a simple conversational assistant can be integrated into a web or CLI app.

## Features
- Ask about courses, departments, and schedules
- Find campus services and contacts
- Small, extensible codebase for learning and experimentation

## Quick start
1. Clone the repo and open the project folder.
2. Inspect the project type and dependencies:
   - If there's a package.json: run `npm install` then `npm start` (or `npm run dev`).
   - If there's requirements.txt: create a venv, `pip install -r requirements.txt`, then run the main script (e.g., `python app.py`).
3. Check README sections or package scripts for exact commands.
